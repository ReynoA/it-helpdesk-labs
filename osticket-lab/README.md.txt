# osTicket Help Desk Lab

This lab simulates a real-world IT help desk environment using osTicket.

## Scenarios Covered

- Adobe Reader Issue (application troubleshooting)
- Banking Outage (high-priority incident handling)

## Skills Demonstrated

- Ticket lifecycle management
- SLA and priority handling
- End-user communication
- Troubleshooting methodology
